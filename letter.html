A note on why this exists.

pinchfin is being released in the spirit of collaboration.

Commercial real estate has traditionally been guarded, skeptical, and human-capital intensive. That skepticism is not irrational. This is a risk business. Judgment matters. Context matters. Experience matters.

pinchfin does not replace any of that.

It is a structured, modular credit framework built to assist, not substitute, human underwriting and institutional decision-making.

This system was built largely in off hours — late nights, early mornings, and weekends — mostly on a laptop from a couch. There are holes. There are patches. There are places where orchestration could be tighter, cleaner, or more deterministic. I am aware of most of them.

pinchfin is alive. It is evolving.

From a compute and build perspective, this project has operated under real constraints:

  • No enterprise R&D team
  • No billion-dollar budget
  • No dedicated security staff
  • No production-grade infrastructure
  • Limited API integrations relative to what institutional firms use

The last 10–15% of AI orchestration — the "magic" that large model providers make look effortless — is extraordinarily complex. Reproducing that layer independently is capital intensive and non-trivial. This system does not claim to have solved that problem.

What it does offer is structure.

pinchfin has been developed alongside real underwriting workflows. It has been run against:

  • 60+ detailed analyses — ~$3B in review
  • 38+ live loans — strong comparative results
  • 1,700+ loan pipeline — ~$27B in ongoing analysis campaigns

We have reviewed logs, tested against analyst notes, compared outputs to approved structures and sizers, and iterated repeatedly. Improvements are ongoing. Logging is ongoing. Refinement is ongoing.

This is not a finished product. It is a living credit framework.

  • Use it responsibly.
  • Never rely on outputs without human review.
  • Never substitute judgment with automation.
  • Always validate assumptions and data inputs.

If you are a lender, broker, sponsor, analyst, or technologist, I hope this sparks ideas about what may be possible inside your own workflows.

If you see flaws, surface them.
If you see improvements, build them.
If you want to fork it, fork it.
If you want to add APIs, add them.

CRE is a collaborative industry on a transactional basis. Pinchfin provides the opportunity to collaborate openly, and systematically.

Be curious. Be Kind. Be Good.

— Brian Murphy
