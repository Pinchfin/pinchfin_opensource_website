Version history and release notes for the pinchfin credit operating system.

v1.0 2026-02-22

Initial Public Release

First public release of the pinchfin credit operating system.

Platform

  • 24 interconnected modules spanning the full CRE credit lifecycle
  • 229+ specialized skills for underwriting, analysis, and portfolio management
  • 17-phase execution framework with 4 blocking gates
  • 4-segment multi-session execution model with state persistence
  • Credit Soul configuration system for institutional credit philosophy capture

Data Services

  • MCP server with 11 tools across 10 external data services
  • Integrated data services for geographic, demographic, economic, environmental, and property-level intelligence
  • 721 county-to-CBSA mappings for geographic resolution
  • Address validation and geocoding services

Quality Assurance

  • Triple-Agent QA system: Producer, Checker, Resolver
  • Three deployment intensity levels: TRIPLE, DOUBLE, SINGLE
  • Skill Chain Audit for mandatory execution logging
  • Improvement log with continuous learning integration

Output & Reporting

  • 8-document model: Sponsor Analysis, Market Analysis, Comparable Analysis, Rent Roll Analysis, T12 Expense Analysis, Business Plan Analysis, Cash Flow Underwriting Analysis, IC Memo
  • Professional report generation with institutional formatting
  • Dual-scenario cash flow projections with sensitivity analysis

Documentation

  • Project website with platform overview, operations guide, and community pages
  • 10-section MCP Guide outline
  • Terms of service and platform documentation
  • All rights reserved
